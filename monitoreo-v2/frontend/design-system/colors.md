# Colors

## Palette

| Token | Hex / Value | Uso |
|---|---|---|
| `bg-primary` | `#000000` | Background dominante |
| `bg-surface` | `rgba(255,255,255,0.05)` | Cards, áreas elevadas |
| `bg-surface-hover` | `rgba(255,255,255,0.08)` | Hover sobre surfaces |
| `border-default` | `rgba(255,255,255,0.10)` | Separadores, bordes de cards |
| `border-hover` | `rgba(255,255,255,0.20)` | Bordes en hover |
| `text-primary` | `#FFFFFF` | Títulos, CTAs, texto principal |
| `text-secondary` | `rgba(255,255,255,0.80)` | Body text, descripciones |
| `text-tertiary` | `rgba(255,255,255,0.50)` | Labels, placeholders |
| `text-muted` | `#666666` | Metadata, timestamps |

## Gradients

| Nombre | Valor | Uso |
|---|---|---|
| `gradient-hero` | `linear-gradient(to bottom, #000, #111827, #000)` | Hero background vertical |
| `gradient-fade-top` | `linear-gradient(to bottom, transparent, #000)` | Fade sobre imágenes |
| `gradient-fade-bottom` | `linear-gradient(to top, transparent, #000)` | Fade inferior |

## Filosofia

- **Monocromático dark-first**: sin color de acento.
- Todo el contraste viene de variaciones de opacidad del blanco sobre negro.
- Las secciones alternan entre `#000000` y `gray-900` (#111827) para separación visual sin introducir color.
