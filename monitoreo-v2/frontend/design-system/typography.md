# Typography

## Font

**Plus Jakarta Sans** — variable font, cargada via `@fontsource-variable/plus-jakarta-sans`.

Fallback: `system-ui, -apple-system, sans-serif`.

## Scale

| Token | Size | Weight | Line-height | Uso |
|---|---|---|---|---|
| `display` | 38px (2.375rem) | 600 | tight (1.1) | Hero principal |
| `h1` | 32px (2rem) | 600 | tight (1.2) | Section headers |
| `h2` | 30px (1.875rem) | 600 | tight (1.2) | Subsections |
| `h3` | 24px (1.5rem) | 600 | snug (1.3) | Card titles |
| `h4` | 20px (1.25rem) | 500 | snug (1.3) | Subtitles |
| `body-lg` | 18px (1.125rem) | 400 | relaxed (1.6) | Body principal |
| `body` | 16px (1rem) | 400 | relaxed (1.5) | Body default |
| `body-sm` | 14px (0.875rem) | 400 | normal (1.5) | Labels, metadata |
| `caption` | 12px (0.75rem) | 500 | normal (1.4) | Badges, tags |

## Weights

| Weight | Valor | Uso |
|---|---|---|
| Regular | 400 | Body text, descripciones |
| Medium | 500 | Subtitulos, labels activos |
| Semibold | 600 | Headings, CTAs, metricas |

## Notas

- No se usa ExtraBold (800). El peso maximo es Semibold (600).
- Metricas de impacto ("3x", "50%", "+1,500") usan semibold a tamaño display.
- Text color siempre en opacidad: 100% para titulos, 80% para body, 50% para terciario.
